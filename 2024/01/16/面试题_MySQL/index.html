面试题_MySQL

面试自测题

  • 说一下MySQL的架构。
  • 讲一下SQL语句在 MySQL 中的执行过程。
  • MySQL提供了哪些存储引擎。
  • 讲一下 MySQL中存储引擎的 InnoDB 和 MyIsam 的区别。
  • 什么是MySQL事务。
  • 事务中 的 ACID 是指什么。
  • 并发事务会带来什么问题。
  • SQL 事务隔离级别有哪些。
  • 不可重复读和幻读的区别。
  • MySQL的默认隔离是什么，能解决幻读问题吗。
  • InnoDB 事务隔离级别实现原理。
  • 什么是MVCC，有什么用，原理了解吗。
  • 表级锁和行级锁了解吗，有什么区别。
  • 行级锁的使用有什么注意事项。
  • 讲一下共享锁和排他锁。
  • 意向锁有什么作用。
  • MySQL 的 InnoDB 引擎中行锁有几种类型？
  • 索引有什么用？讲一下索引的优缺点。
  • 讲一下索引的底层结构。
  • MySQL 索引：索引为什么使用 B+树？
  • 讲一下这些区别：主键索引和二级索引、聚集索引和非聚集索引。
  • 讲一下 联合索引、覆盖索引。
  • 建立索引的时候会遵守最左前缀匹配原则，讲一下这个。
  • 创建索引需要注意什么。
  • MySQL 日志：常见的日志都有什么用？
  • 慢查询日志有什么用。
  • binlog 是什么，它主要记录了什么，有什么用。
  • 讲一下 redolog ；同时说一下 redo log 如何保证事务的持久性。
  • 页修改之后为什么不直接刷盘。
  • binlog 和 redolog 有什么区别。
  • undo log 如何保证事务的原子性。
  • 讲一下SQL优化常见手段。
  • 讲一下慢查询问题排查。
