1、项目介绍

项目学习目标：去熟练一下微服务应用体系的构建，在这个体系上通过配置环境Docker + Nginx + SSL；开发出鉴权、认证、微信公众号、企业微信、支付宝交易等模块，完善体系的物料服务。

将核心的基础物料【物料指SDK和服务】通过 API 的方式与具体的物料服务模块解耦。

那么就开始吧。

技术栈：SpringBoot、MyBatis、MySQL、Redis

开发工具： IntelliJ IDEA、WebStorm、Docker、Protainter、Nginx、Git、Maven、Navicat、SSH工具

前端：Next.js、Typescript

课程目录


开发流程：

    1. 基础设置的服务搭建
    1. 接口鉴权开发
    1. API-SDK 的开发，网页的简单开发
    1. ChatGPT-WEB-UI -> API-SDK -> 鉴权 -> OpenAI 的使用
    1. ChatGPT-WEB-UI 流程🏃🏻跑通，逐步扩展其他服务模块。让业务与场景结合
    • 关注公众号、公众号回复、企业微信机器人、交易支付购买授权Token
    • 让 ChatGPT-WEB-UI 与各个模块结合使用。

推荐工具：

  • Termius：云服务器链接工具，自带一套 SFTP 工具，很好用。
  • IDEA 插件：Sequence Diagram - 用于方法上右键查看代码流程的，主要帮助大家理解代码。
  • Docker - 可本地安装，不需要非得购买云服务器，也可以满足学习诉求。
  • natapp - 内网穿透工具，相当于你开启后，可以分配给你一个域名，让外网访问到你的服务
