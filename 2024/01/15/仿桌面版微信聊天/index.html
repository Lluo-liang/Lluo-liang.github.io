仿桌面版微信聊天

学习项目地址： https://wx.zsxq.com/dweb2/index/columns/48411118851818

项目地址：

  • https://gitcode.net/KnowledgePlanet/Netty/IM

1、项目介绍

功能：仿桌面版微信聊天

技术栈：JavaFx、Netty4.x、SpringBoot、Mysql

2、运行项目

下载项目中的三个项目，先加载并install，顺序分别是：itstack-naive-chat-server → itstack-naive-chat-ui → itstack-naive-chat-client

直接启动 Applicatioin 类就行，改一下数据库连接配置。

3、UI端

从这一章开始介绍一下实现的功能，并结合文档进行阐述。

UI 端是使用 JavaFx 来进行构建的一个桌面应用，主要的页面是包括等登录界面、聊天界面、对话弹出框、好友栏(参考桌面微信页面布局）。

建议先按照学习项目地址中的 业务项目 → IM 实战：Netty+JavaFx实战：仿桌面版微信聊天 → 课程目录 → 第一章 UI 开发

全部先自己看一遍

UI 端是项目工程看具体使用是构建后，打成一个 Jar 包，供客户端引入依赖使用；达到 UI 与 业务分离。

4、架构设计

使用到了四层架构模型DDD

  • 应用层 Application
  • 领域层 domain
  • 基础层 infrastructrue
  • 接口层 interfaces

将代码工程进行了一定的分层

5、功能实现

主要功能：登录、添加好友、对话通知、消息发送、断线重连。
