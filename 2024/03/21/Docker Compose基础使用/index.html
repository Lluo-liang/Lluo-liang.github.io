Docker Compose基础使用

今天学习一下 Docker Compose 的基础使用

介绍：Compose 项目是 Docker 官方的开源项目，负责实现对 Docker 容器集群的快速编排。

Compose 允许用户通过一个单独的 docker-compose.yml 模板文件（YAML 格式）来定义一组相关联的应用容器为一个项目（project）

Compose 中有两个重要的概念：

  • 服务 (service)：一个应用的容器，实际上可以包括若干运行相同镜像的容器实例。
  • 项目 (project)：由一组关联的应用容器组成的一个完整业务单元，在 docker-compose.yml 文件中定义。

卸载操作：

如果是 windows 或者 macOS ，需要卸载一下 Docker Desktop。

如果 Docker Compose 是作为一个单独的二进制文件安装的：

  1. 找到 Docker Compose 的安装位置。如果你不确定其安装位置，可以通过运行 which docker-compose（在 Linux 或 Mac 上）或 where docker-compose（在 Windows 上）来找到它。

  2. 删除 Docker Compose 二进制文件。一旦你知道了 Docker Compose 的安装位置，你可以手动删除该文件。在 Linux 或 Mac 上，通常可以使用 sudo rm 命令来删除：

2、如何使用

Compose 使用的三个步骤：

  • 使用 Dockerfile 定义应用程序的环境。
  • 使用 docker-compose.yml 定义构成应用程序的服务，这样它们可以在隔离环境中一起运行。
  • 最后，执行 docker-compose up 命令来启动并运行整个应用程序。
