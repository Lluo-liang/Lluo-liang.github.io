1、启动Nginx

Windows 下启动 nginx

在下载 nginx 后，在nginx安装目录的绝对路径的框框内输入 cmd

直接输入 nginx 或者 start nginx，然后回车，就可以启动nginx

在浏览器地址栏输入 localhost:80，然后再回车, 查看是否有 nginx 访问主页，如果有，则启动成功。

一般访问问题是端口占用，可以改一下 nginx 访问端口，或者杀掉对应进程。
