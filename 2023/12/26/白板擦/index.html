日常计划

  • 1、修改为早点休息,早上早点起床,可以尝试固定两小时去做一些事情,最好是工作之外的事情,包括学习或者其他
  • 2、晚上时间不强制要求,早起培养起来
  • 3、减少无效社交时间, 随缘社交, 增加一些随机性,尽可能还是去接触一些实际的人,增加自己的一些幽默感和阅历
  • 4、计划好自己的周末, 并有一些地方可以去
  • 5、注意自己的卫生, 注意自己的穿着搭配, 及时整理自己的房间,和自己宽解, 好好休息并加油.
  • 6、整理业务或者某个单独课题的话,可以单独开一个工作空间(如果不开就整理一下之前的内容,保持持续跟进)

注意事项

  • 人生也许没有很多犹豫，既然选择了就去做，尽可能减少摇摆

  • 多去锻炼自己的表达能力，多去沉淀自己

  • 固定两个小时学习，同时期间不接触手机等设备，早点休息

  • 如果你真的想学习的话，就把小说卸载了，你抗拒不了他，且他带不了什么有效情绪。（认识到自己抵制不了就不要去接触）

  • 后续打算搞多一点这个MAC电脑

  • 长期打算

    • 1、软考
    • 2、认识一些有趣的人,但是自己需要努力,或者说自己需要有一个还算过得去的生活状态
    • 3、尝试赚钱,工作之外的钱
    • 4、持续学习,这个很重要
    • 5、考研(去做)
    • 6、学习 AI 方向的一些内容,主要是 Java + AI. 赛道, 自己还是太小的投入,跟实际的比较差距较大,期望能作为某个点去改变一些东西
    • 7、未来的城市方向: 珠海(暂时不改了), 希望变得优秀, 同时经营自己的生活状态,学会旅行和拍照,学会搭配,学习AI和 JAVA 路线,努力考研和证书, 多交一些朋友,认真的去改变自己.
